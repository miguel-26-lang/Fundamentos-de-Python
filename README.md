# Calculadora de Índice de Masa Corporal (IMC)
## Descripción del programa
Este programa fue desarrollado en Python y permite calcular el Índice de Masa Corporal (IMC) de una persona a partir de su peso y estatura.
El programa solicita al usuario los siguientes datos:
- Nombre
- Apellido paterno
- Apellido materno
- Edad
- Peso
- Estatura
Posteriormente calcula el IMC utilizando la fórmula:
IMC = peso / estatura²
El programa incluye validación de datos para evitar:
- Campos vacíos
- Espacios en blanco
- Datos incorrectos en campos numéricos
Finalmente muestra la clasificación del IMC según los rangos establecidos.
## Cómo ejecutar el programa
1. Descargar el archivo `imc.py`
2. Abrir una terminal
3. Ejecutar:
python imc.py
